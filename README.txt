Bíblia Santa — Refúgio Interior (pacote completo)
Arquivos incluídos:
- index.html : página principal (cliente)
- style.css  : estilização
- script.js  : lógica front-end (conteúdo organizado por capítulos)
- nightmare.png : placeholder (imagem de capa)

Como usar:
1) Descompacte o .zip e abra index.html em um navegador moderno.
2) Para publicar no GitHub Pages: crie um repositório, envie os arquivos para a branch 'main' e ative GitHub Pages (branch main / root).
3) O botão 'Baixar capítulo' permite salvar o capítulo selecionado em .txt.

Observações:
- O conteúdo (textos, salmos e orações) foi inserido conforme solicitado. Se quiser adicionar, editar ou reorganizar capítulos, me diga e eu atualizo os arquivos.
